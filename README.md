# Adarsh Portfolio

A modern and responsive developer portfolio built with React, Tailwind CSS, and Framer Motion.

Designed to showcase projects, skills, services, testimonials, blog articles, and contact information through a clean and engaging user experience.

## ✨ Features

* Modern dark-themed UI
* Responsive layout
* Smooth scrolling navigation
* Active section highlighting in navbar
* Animated page sections using Framer Motion
* Reusable React components
* Project showcase section
* Services section
* About section
* Client testimonials
* Blog preview section
* Contact form UI
* Hover effects and micro-interactions

## 🛠️ Tech Stack

* React
* Tailwind CSS
* Framer Motion
* React Icons
* Vite

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/portfolio.git
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
